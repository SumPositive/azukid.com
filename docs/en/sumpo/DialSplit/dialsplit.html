DialSplit User Guide

Dial-style bill splitting

DialSplit helps you split a total amount by adjusting people and amounts with tactile dial controls.

Set the total and panel values, then DialSplit automatically computes Panel A's amount.

Basic Usage

Set the total amountUse the total dial on the top panel, or tap the amount text to enter with the numpad.
Set people countsUse the left dial on each A/B/C/D panel. Panel A requires at least 1 person.
Set amountsAdjust amounts on B/C/D with the right dial. Panel A amount is auto-calculated.
Change dial stepUse Amount Dial Step to switch increments (JPY in Japanese UI, USD in English UI).
Tune panel styleAdjust panel brightness, text color, and tone from the Panel Style section.

Specifications

People range A: 1-99, B/C/D: 0-99
Amount range 0-999,900
Input methods Dial input + direct numpad input
Languages Japanese / English

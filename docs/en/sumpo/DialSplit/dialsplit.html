Divigo User Guide

A visual guide built around screen structure

Divigo helps you decide bill splits by adjusting people counts and amounts with tactile dials

This guide is organized around visual structure first, so you can understand the main screen, locks, settings, and currency behavior at a glance

4 Tiers Adjust A / B / C / D independently
2 Locks People lock and full control lock
Locale Aware Currency display follows your device

Main screen

Tap a marker to open its explanation

Total people Check the total number of people
Total panel lock Lock the people dials to prevent accidental changes
Panel A In Panel A, specify only the number of people. The amount is calculated automatically
Total amount Enter the total payment first. Tap the number to enter it with the numpad
Amount dial step Switch the amount dial increment. Press and hold to set its value
Panels B / C / D In Panels B / C / D, specify the people and amount. Any difference is assigned to Panel A
Panel style Customize the look of this screen. It does not affect the bill split
Settings Open the settings screen
Full control lock Lock everything to prevent accidental changes

Usage flow

1 Enter the total

Set the bill total from the top panel with the dial or the numpad

2 Set people counts

Adjust people for A / B / C / D with the left-side dials

3 Adjust amounts

Tune B / C / D amounts and let Panel A update automatically

4 Lock if needed

Freeze people only or freeze all controls to avoid accidental changes

Locale-aware currency handling

Japanese / Japan

¥12,000 Dial step examples: 1 / 10 / 100 / 500 / 1000

Handled as an integer currency. Step labels are shown without a currency symbol

English / United States

$12.00 Dial step examples: 0.01 / 0.10 / 1.00 / 5.00 / 10.00

For two-decimal currencies, values are stored using the smallest currency unit

English / Kuwait

KWD 1.000 Dial step examples: 0.001 / 0.010 / 0.100 / 0.500 / 1.000

Three-decimal currencies are also supported without mismatch between display and internal units

Calclin User Guide

Calculator memo app with paper-roll history

Tap the keys like a calculator to enter a calculation. Tap [=] at the end to get the answer.

Your calculation history stays visible like a paper roll. Use the Calculator/Formula button in the input row to switch calculation style, and the PDF button to export the current roll.

Multiple calculation frames, notes, appearance mode, decimal digits, rounding, and keyboard customization can be adjusted in Settings.

Specifications

Calculation Engine

Engine C/C++ fixed-point decimal. Internal calculations are stored in BCD arrays, and the significant-digit count can be changed by definition values. The current app setting uses 60 digits.
Calculation mode
  • ★ Calculator mode — left-to-right evaluation (5+5×2=20), like a physical calculator
  • Formula mode — operator precedence respected (5+5×2=15), with parentheses, √, ∛
Operations Four arithmetic operations
Calc fields Reverse Polish style, up to 3 fields
History list Up to 100 entries. Oldest entries are deleted first (circular).

Rounding modes
  • Round up — always toward positive infinity
  • Round toward +∞ — always increases
  • Round half away from zero [JIS Z 8401 Rule B]
  • ★ Round half to even [JIS Z 8401 Rule A] — even rounding / banker's rounding
  • Round half toward +∞
  • Round toward −∞ — always decreases
  • Truncate — always toward zero

モチメモ　取扱説明

持ち物チェックリストアプリ

旅行やイベントに役立つ持ち物メモです。自由自在に編集できます。

旅行や出張などにご利用ください。重量集計により、体力に合わせた荷造りができます。趣味の用品管理などにもお役立てください。

基本操作の流れ

1 パックを作る

旅行やイベントごとに、持ち物をまとめるパックを作成します。

2 アイテムを登録する

必要数、在庫数、重量などを入力して、持ち物リストを整えます。

3 チェックする

準備状況を確認し、不足数や重量を見ながら荷造りします。

4 共有・保管する

必要に応じて、パックの書き出しやバックアップを行います。

⚠️ データは暗号化しないので、個人情報や秘密情報を入れないでください。

Undo について

操作手間の最小化を目指しました。よくある 「確認」ダイアログはなく、編集後の 「保存」ボタンもありません。「削除」は確認なしに即削除します。

間違えたときは 「Undo ↩︎」 ボタンで前の状態に戻せます。最大 10 回前まで戻れます。Redo で取り消しを戻すことも可能です。

アイテム縦覧

グループ一覧画面下部の「アイテム縦覧」ボタンから、パック内の全アイテムをグループをまたいで一覧表示できます。出発前の最終チェックや荷物の見直しに便利です。キーワード検索にも対応しています。

並べ替え

未 ✔︎ 順 未チェックのアイテムを先頭にまとめます。出発前の最終確認に最適です。
不足個数 在庫数が必要数に届いていないアイテムを不足量の多い順に表示します。
不足重量 不足分の重量（不足個数 × 単重）が大きい順に表示します。
在庫重量 現在の在庫重量（在庫数 × 単重）が大きい順に表示します。重量オーバー時の見直しに役立ちます。

その他

キーワード検索 名前・メモを対象にリアルタイムで絞り込みます。
常時並べ替え オンにすると、チェック・在庫の変更に合わせてリストが自動で並び直ります。

数量入力

個重量・在庫数・必要数はそれぞれ 2 とおりの方法で入力できます。

入力方法

ダイアル 各フィールド右側のダイアルを左右にドラッグして値を増減します。スタイル（見た目）と感度（ピッチ・スムージング・慣性）はダイアル設定でカスタマイズできます。
テンキー フィールドの数値部分をタップすると専用テンキーシートが開きます。直接数値を入力できます。

ダイアル設定

スタイル Legacy / Midnight / Brass / Ocean / Shape / Varnia / Chrome / Hairline / Rubber の 9 種類から選べます。
感度プリセット Fine / Mild / Standard / Light / Fast の 5 段階から選べます。
詳細調整 ピッチ・スムージング・慣性開始速度をスライダーで細かく調整できます。

パック共有・バックアップ

書き出し

パック単体 パック画面のメニューから「書き出す」を選ぶと、そのパック 1 件を .packlin ファイルとして書き出せます。AirDrop・メール・メモアプリなど任意の方法で共有できます。
全パックまとめて 設定画面の「全パックを書き出す（バックアップ）」から、全パックを 1 つの .packlin ファイルとして書き出せます。機種変更やデータ保全にご利用ください。

読み込み

操作 設定画面の「パックを読み込む」からファイルを選択します。単体パックとバックアップのどちらも同じ操作で読み込めます。
重複の扱い バックアップ復元では ID が一致するパックを上書きします。単体パックの共有では名前が一致するパックを上書きします。一致しない場合は新規追加されます。
ファイル判別 拡張子ではなく JSON の内容で単体／バックアップを自動判別します。

AI 生成（チャッピー）オプション・有料

旅行先やイベントの目的を伝えると、AI がパックの内容を提案してくれます。提案されたパックはそのまま保存でき、あとから自由に編集できます。既存パックを AI に渡して修正を依頼することもできます。

1 回の生成につき AI 利用券 1 枚が必要です。利用券はアプリ内購入（1 枚・5 枚）または広告視聴で入手できます。

利用方法

新規作成 パックリスト右上の ✦ ボタンから「チャッピーに作ってもらう」を選択して起動します。
既存パック修正 パック編集画面の ✦ ボタンから「チャッピーに修正してもらう」を選択します。現在のパック内容が AI に渡され、追記・修正の依頼ができます。
要望テキスト 最大 1,000 文字。旅行先・日数・目的・季節など自由に記述してください。

AI 利用券の入手方法

アプリ内購入 1 枚・5 枚のセットを購入できます。
広告視聴 設定画面の「広告を見て寄付」から動画広告を視聴することでも入手できます。

設定

表示

表示モード 初心者：ヘッダーに操作説明を表示します。達人：説明を省いてコンパクトに表示します。
外観モード システム：端末の設定に従います。ライト：常にライトモード。ダーク：常にダークモード。
明細表示 行ごとの表示行数を切り替えます。最小：名称 1 行・メモ非表示　1 行：名称 1 行・メモ 1 行　2 行：各 2 行　3 行：各 3 行。

操作

新規追加位置 新しいアイテム・グループ・パックをリストの 先頭 または 末尾 に追加するか選べます。

重量

必要重量を表示 必要数ぶんの重量（必要数 × 単重）を表示します。
重量計 1000g から kg 表示 合計重量が 1,000g 以上になると「1.0 kg」形式に切り替えます。

チェックと在庫の連動

チェック ON で在庫を満たす チェックをオンにしたとき、在庫数を必要数まで自動で増やします。
チェック OFF で在庫を 0 にする チェックをオフにしたとき、在庫数を 0 にリセットします。

Item 個として扱う物品

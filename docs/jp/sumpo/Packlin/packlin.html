モチメモ (Packlin)

旅行やイベントに役立つ持ち物メモです。自由自在に編集できます。

Undo

操作手間の最小化を目指しました。よくある「確認」操作はありません、編集後の「保存」ボタンもありません。「削除」は確認なしに削除します。もし、間違ったならば「Undo ↩︎」ボタンで前の状態に戻せます。最大10回前まで戻せます

機能

  • アイテム毎にチェックして全体のチェック状況を確認できる
  • 必要数と在庫数を記録できる
  • アイテムの重量(g)を登録して総重量を確認できる
  • ドラッグ＆ドロップ操作で並べ替えができる
  • アイテムを他のグループやパックへ移動や複製ができる
  • グループを他のパックへ移動や複製ができる
  • Undo：最大10回前の状態に戻せる
  • Redo：最大10回Undo前の状態に戻せる

仕様・諸元

  • DATABASE：SwiftData (sqlite3)
  • 数量制限（最大）：ほぼ無制限なのですが、動作検証のため制限しています。もし、拡張したい活用法があれば教えてください
    • パック：30
    • グループ：30
    • アイテム：100
    • 個重量：999,999 (g)
    • 在庫数：999
    • 必要数：999
    • 名称：200文字
    • メモ：200文字
  • データ構造：下記の三層で記録します。Itemが最小単位になります 
    App
├─ Pack（パック：バッグやリュック、トランクや倉庫など）
│   ├─ Group（グループ：ポーチやボックスなどで小分けしましょう）
│   │   ├─ Item（アイテム：個として扱う物品です）
│   │   └─ Item
│   └─ Group
│       ├─ Item
│       └─ Item
└─ Pack
    ├─ Group
    │   ├─ Item
    │   └─ Item
    └─ Group
        ├─ Item
        └─ Item
  • Version x.y.z.#：アプリ製造番号：各値は0から始まり下記の要領でカウントアップします
    • x：大幅改良、全面刷新
    • y：機能追加・変更
    • z：微調整、不具合修正
    • #：ビルド連番
  • Support：サポート識別番号：8〜12桁くらいの英数字
    • 個人を特定しない（できない）番号です
    • サーバ内にあるログを調査する必要が生じた時に使用します

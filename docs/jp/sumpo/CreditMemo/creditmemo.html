クレメモ　取扱説明

後日引き落とし管理アプリ

クレメモは、あとで口座から引き落とされる決済を記録して管理するためのアプリです。クレジットカード利用、クレジット連携の電子マネーチャージ、N日後引き落とし、指定日引き落とし などを管理できます。

基本操作の流れ

1 決済を記録する

金額を入力し、必要に応じてラベルやタグを付けて保存します。

2 決済手段を選ぶ

カードや後払い手段を選び、引き落とし予定を決めます。

3 引き落としを確認する

未払と支払済みを確認し、口座ごとの予定を把握します。

4 見直し・保管する

履歴を修正したり、JSON でバックアップしたりできます。

⚠️ データは暗号化しないので、個人情報や秘密情報を入れないでください。

できること

新しい決済をすばやく入力金額を先に入れて保存し、細かい情報はあとから補えます。
履歴をあとから修正保存後の明細を開いて、ラベル・タグ・決済手段などを更新できます。
引き落とし予定をまとめて確認直近の引き落とし合計と、未払・済みの状態を一覧できます。
口座別に集計同じ引き落とし日でも、口座ごとに分けて把握できます。
決済手段ごとに支払日を管理締日/支払日型と N日後型に対応します。
タグを複数付与達人モードではタグ別の整理や再編集ができます。
JSON でエクスポート/インポートバックアップや他端末への移行に使えます。
表示や保存後動作を調整文字サイズ、起動時画面、保存後の戻り先などを設定できます。

画面構成

新しい決済 新規の決済明細を入力します。起動時にこの画面を開く設定もできます。
履歴 保存済みの明細を一覧します。すべて、未選択あり、決済手段別で絞り込めます。
引き落とし状況 引き落とし前と引き落とし済みをまとめて表示します。タップで未払/済みを切り替えられます。
決済手段 カード名や後払い手段、締日/支払日、引き落とし口座を登録します。
引き落とし口座 引き落とし先の口座名を管理します。決済手段ごとに関連付けできます。
タグ 用途や分類のためのタグです。達人モードで利用できます。
設定 表示、保存後動作、バックアップ、サポート情報をまとめています。

新しい決済を入力する

まずは 金額を入れて保存 してください。決済手段、ラベル、タグはあとから補完できます。初心者モードではこの流れに沿った案内を表示します。

基本入力

金額 専用テンキーで入力します。保存に必須です。
利用日 カレンダーから選択します。引き落とし予定日は、決済手段の設定に基づいて自動計算されます。
決済手段 カードや後払い手段を選びます。未選択のまま保存し、あとで履歴から補うこともできます。
引き落とし口座 選んだ決済手段に口座が未設定のときだけ、この画面で口座を選べます。

補足入力

決済ラベル 店名や用途など、一覧で見分けやすい短い名前を入れます。過去入力から候補も表示されます。
タグ 達人モードで複数選択できます。あとで変更や追加もできます。
繰り返し 達人モードの一括払いで使えます。対象の決済を引き落とし済みにしたとき、翌月など指定した間隔の新しい決済を自動追加します。
メモ 自由記入欄です。あとから見返したい補足情報を残せます。

似た決済の流用

類似候補 新規入力では、金額を入れると過去 18 か月の類似決済候補が表示されます。タップするとラベル、決済手段、タグなどをまとめて流用できます。

履歴を見直す

並び順 通常は直近入力順です。利用日順ではありません。
フィルター すべて、未選択あり、決済手段別で絞り込めます。未選択ありでは、決済手段未設定・ラベル未設定・タグ未設定の明細を優先表示します。
編集 一覧の明細を開くと、内容の修正や削除ができます。すでに引き落とし済みの明細は、金額・利用日・決済手段など一部の中核項目が固定されます。

引き落とし状況

引き落とし状況では、未払の予定と引き落とし済みを 1 つの流れで確認できます。引き落としを確認したら、その行をタップして 済み に切り替えてください。必要なら未払へ戻すこともできます。
未払の区分 当日以降の予定と、引き落とし日を過ぎた未払を分けて表示します。過去未払があるときは切り替えセグメントが表示されます。
直近合計 トップ画面には、設定した期間ぶんの直近引き落とし合計を表示します。期間は 1〜20 日または 1 か月で変更できます。
口座別集計 同じ日付でも口座が異なる請求は別の引き落としとして集計します。
明細表示 引き落とし行を開くと、その引き落としに含まれるカード別請求や元の決済明細を確認できます。

決済手段・口座・タグ

決済手段

用途 後日口座から引き落とされる決済手段だけを登録します。PayPay ならクレジット連携利用や残高チャージが対象で、チャージ済み残高の支払いは対象外です。
請求方式 締日/支払日型と N日後型を選べます。締日が月末の場合は月末指定もできます。
口座関連付け 決済手段ごとに引き落とし口座を 1 つ設定します。変更すると、その決済手段に属する過去の請求や引き落としの所属も再計算されます。

口座・タグ

口座 引き落とし先の口座名を管理します。プリセットから名前を引用することもできます。
タグ タグ編集画面では、そのタグが付いた決済明細を一覧で見返せます。

バックアップと設定

JSON バックアップ

エクスポート 全データを JSON ファイルとして書き出せます。保存向けのコンパクト形式と、人が読みやすいプリティ形式を選べます。
インポート 書き出した JSON を読み込めます。同じデータを再読込しても、置換ではなく重複しない形で更新します。
古い履歴の整理 データ量が増えすぎた場合は、設定から古い決済を整理できます。運用上は 3 年程度を目安に保つ想定です。

表示と操作

初心者/達人 初心者は説明を多めに表示し、達人ではタグや繰り返しなどの詳細項目を有効にします。
文字サイズ 自動にすると iOS の文字サイズ設定に追従します。
起動時の動作 アプリ復帰時に「新しい決済」を自動で開く設定ができます。
保存後の動作 前画面に戻る、連続入力する、同じ日付と決済手段を保って連続入力する、履歴を表示する、から選べます。

基本仕様

対象 後日口座から引き落とされる決済
対応 OS iOS 18 以降
金額上限 99,999,999
利用日範囲 2000/1/1 〜 2100/12/31
